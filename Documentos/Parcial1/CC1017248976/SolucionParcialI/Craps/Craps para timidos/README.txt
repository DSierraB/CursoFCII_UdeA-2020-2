ESte codigo corresponde a la versión CRAPS para tímidos, en el cual se presenta el juego por diversión, libre de apuestas.


Para correr y compilar este código
$ g++ CRAPS.cpp CrabsGame.cpp -o game.out 
$ ./game.out 

los archivos en esta carpeta corresponden a:
Crap.h -> En este archivo se encuentra todo lo relacionado a la clase
CrabsGame.cpp -> En este archivo se especifican las funciones de la clase 
CRAPS.cpp -> En este archivo se encuentra el archivo Main()
