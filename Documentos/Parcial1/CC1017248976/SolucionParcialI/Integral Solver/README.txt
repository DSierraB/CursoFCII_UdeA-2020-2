Para compilar y correr este codigo

$ g++ IntegralSolver.cpp TrapezoidRule.cpp -o Integral.out
$ ./Integral.out 

Los archivos de esta carpeta corresponden a:
TrapezoidalClass.h -> Contiene la clase
TrapezoidRule.cpp -> Contiene todas las funciones, incluyendo el algoritmo de la regla trapezoidal para integrar
IntegralSolver.cpp -> Contiene la funcion Main
